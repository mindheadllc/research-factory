# Spike Assessment

## PURPOSE

A one-page rapid feasibility evaluation for a research idea. This is the output of Tier 1 (Spike), designed to kill bad ideas in 1-2 weeks before any infrastructure is created. Only the COO and Red Team are involved. No file tree, no git, no task tracking — just this document.

**If the idea dies here**, this document is the only artifact. File it for future reference or discard it.

**If the idea escalates to Proof of Concept**, this document becomes input to Phase 0.
